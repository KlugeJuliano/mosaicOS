# Tools

`tools/lab` contains the Milestone 1 L4Re laboratory scripts:

- `setup.sh`: install Ubuntu/Debian dependencies and run the full build.
- `build-kernel.sh`: fetch and build Fiasco.OC.
- `build-l4re.sh`: configure and build L4Re userland.
- `build-hello.sh`: build the hello and IPC experiments.
- `run-qemu.sh`: boot a lab entry in QEMU.

See `docs/lab-setup.md` for the full workflow.
